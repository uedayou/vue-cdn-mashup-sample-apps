# vue-cdn-mashup-sample-apps

CDN版Vue.jsと各オープンデータファイルやWeb APIでマッシュアップしたサンプルアプリを公開しています。

## 公開アプリ

Vue2 と Vue3 + ES Modules の2種類あります。
